飞机射击游戏 - 使用说明
========================================

🎮 游戏简介
这是一个使用Python和Pygame开发的飞机射击游戏，具有完整的游戏功能和美观的界面。

📁 项目文件结构
airplane_shooter_game/
├── main.py              # 游戏主入口
├── game.py              # 游戏主类（管理游戏循环）
├── player.py            # 玩家飞机类
├── enemy.py             # 敌机类
├── bullet.py            # 子弹类
├── config.py            # 配置文件（常量设置）
├── requirements.txt     # Python依赖
├── README.md           # 详细说明文档
├── run_game.bat        # Windows启动脚本
├── run_game.sh         # Linux/Mac启动脚本
├── install_deps.py     # 依赖安装脚本
├── test_structure.py   # 项目结构测试
└── assets/             # 资源文件夹（预留）

🚀 快速开始

1. 安装Python 3.6或更高版本
2. 打开终端/命令提示符，进入游戏目录
3. 安装依赖：pip install pygame
4. 运行游戏：python main.py

或者使用启动脚本：
- Windows: 双击 run_game.bat
- Linux/Mac: 在终端运行 ./run_game.sh

🎯 游戏控制
← → ↑ ↓ : 控制飞机移动
空格键   : 发射子弹
R键      : 重新开始游戏
ESC键    : 退出游戏

🏆 游戏规则
1. 使用方向键控制飞机躲避敌机
2. 按空格键发射子弹摧毁敌机
3. 每摧毁一个敌机获得10分
4. 玩家有3条生命，被敌机撞到会减少生命
5. 生命值为0时游戏结束
6. 尽可能获得高分！

🛠️ 自定义设置
如需修改游戏参数，请编辑 config.py 文件：
- 屏幕大小和帧率
- 玩家和敌机属性
- 颜色和视觉效果
- 游戏难度设置

🐛 故障排除
1. 确保已安装Pygame：pip install pygame
2. 检查Python版本：需要Python 3.6+
3. 如果遇到问题，查看错误信息并按提示操作
4. 可以按R键重新开始游戏

📞 支持
如果遇到问题，请检查：
1. 是否正确安装了所有依赖
2. Python版本是否符合要求
3. 文件是否完整无损坏

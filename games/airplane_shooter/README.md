# 飞机射击游戏

一个使用Python和Pygame开发的飞机射击游戏，具有美观的界面和完整的游戏功能。

## 🎮 游戏特性

- **玩家控制**：使用方向键控制飞机移动，空格键发射子弹
- **敌机系统**：随机生成的敌机从屏幕上方出现并向下移动
- **碰撞检测**：子弹击中敌机、敌机撞击玩家
- **得分系统**：摧毁敌机获得分数
- **生命系统**：玩家有3条生命
- **粒子效果**：爆炸和特效
- **游戏状态**：开始界面、游戏进行、游戏结束
- **重新开始**：游戏结束后可以重新开始

## 🚀 运行要求

- Python 3.6+
- Pygame 2.5.2

## 📦 安装和运行

### 方法1：直接运行（推荐）

```bash
# 进入游戏目录
cd airplane_shooter_game

# 安装依赖
pip install pygame

# 运行游戏
python main.py
```

### 方法2：使用requirements.txt

```bash
cd airplane_shooter_game
pip install -r requirements.txt
python main.py
```

## 🎯 游戏控制

| 按键 | 功能 |
|------|------|
| ← → ↑ ↓ | 控制飞机移动 |
| 空格键 | 发射子弹 |
| R键 | 重新开始游戏 |
| ESC键 | 退出游戏 |

## 🏆 游戏规则

1. 使用方向键控制飞机躲避敌机
2. 按空格键发射子弹摧毁敌机
3. 每摧毁一个敌机获得10分
4. 玩家有3条生命，被敌机撞到会减少生命
5. 生命值为0时游戏结束
6. 尽可能获得高分！

## 🎨 游戏界面

- **深蓝色星空背景**：带有移动的星星效果
- **彩色游戏元素**：玩家飞机（青色）、敌机（红色/橙色/紫色）、子弹（黄色）
- **粒子特效**：爆炸时的粒子效果
- **UI显示**：分数、生命值、敌机数量
- **状态界面**：开始界面和游戏结束界面

## 🛠️ 技术特点

1. **面向对象设计**：每个游戏对象都是独立的类
2. **模块化结构**：代码分离清晰，易于维护
3. **碰撞检测**：使用pygame的矩形碰撞检测
4. **状态管理**：清晰的状态机管理游戏流程
5. **视觉效果**：粒子效果、渐变颜色、动画
6. **错误处理**：完善的异常处理机制

## 🔧 自定义设置

可以在 `config.py` 中修改游戏参数：

- 屏幕大小和帧率
- 玩家和敌机的属性
- 颜色和视觉效果
- 游戏难度设置

## 🐛 故障排除

如果遇到问题：

1. **确保已安装Pygame**：`pip install pygame`
2. **检查Python版本**：需要Python 3.6+
3. **查看错误信息**：游戏会显示详细的错误信息
4. **重启游戏**：按R键重新开始

# 飞机大战游戏 (Plane War Game)

一个使用Python和Pygame开发的2D飞机射击游戏。

## 游戏特点

- 🎮 流畅的飞机控制
- 🔫 自动射击系统
- 👾 多种敌机类型
- 💥 华丽的爆炸效果
- ⭐ 道具系统（生命恢复、护盾、加速等）
- 📊 分数系统和难度递增
- 🎨 精美的UI界面

## 系统要求

- Python 3.6+
- Pygame 2.0+

## 安装方法

### 1. 安装Python
确保已安装Python 3.6或更高版本。

### 2. 安装Pygame
```bash
pip install pygame
```

或者使用以下命令检查是否已安装：
```bash
python -c "import pygame; print(f'Pygame版本: {pygame.version.ver}')"
```

## 运行游戏

### 方法1：使用启动脚本（推荐）
```bash
cd plane_war_game
python run_game.py
```

### 方法2：直接运行主文件
```bash
cd plane_war_game
python main.py
```

### 方法3：在Windows上双击运行
双击 `run_game.py` 或 `main.py` 文件（如果已关联Python）。

## 游戏控制

| 按键 | 功能 |
|------|------|
| **方向键** 或 **WASD** | 移动飞机 |
| **空格键** | 射击 |
| **ESC键** | 退出游戏 |
| **R键** | 重新开始游戏（游戏结束后） |

## 游戏规则

1. **目标**：消灭尽可能多的敌机，获得高分
2. **生命值**：玩家初始有100点生命值，被敌机或子弹击中会减少
3. **分数系统**：
   - 消灭普通敌机：100分 × 难度系数
   - 拾取道具：50分
4. **难度系统**：游戏时间越长，难度越高
5. **道具系统**：
   - 生命恢复：恢复生命值
   - 护盾：获得临时护盾
   - 加速：提高移动速度
   - 射速提升：提高射击频率

## 开发说明

### 自定义游戏设置
修改 `config.py` 文件可以调整游戏参数：
- 窗口大小
- 游戏难度
- 敌机生成频率
- 玩家属性等

### 添加新功能
1. **新敌机类型**：在 `enemy.py` 中添加新类
2. **新道具类型**：在 `powerup.py` 中添加新类
3. **新武器系统**：在 `bullet.py` 中添加新子弹类型

## 故障排除

### 常见问题

1. **无法导入Pygame**
   ```
   ModuleNotFoundError: No module named 'pygame'
   ```
   **解决方法**：运行 `pip install pygame`

2. **游戏窗口无法显示**
   **解决方法**：确保已安装正确的Pygame版本

3. **游戏运行缓慢**
   **解决方法**：降低游戏帧率（修改config.py中的FPS值）

4. **字体显示问题**
   **解决方法**：游戏会自动使用系统默认字体

### 获取帮助
如果遇到问题，请检查：
1. Python和Pygame是否正确安装
2. 所有游戏文件是否完整
3. 控制台输出的错误信息

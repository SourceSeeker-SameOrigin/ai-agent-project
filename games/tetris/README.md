# 俄罗斯方块游戏

一个使用Python和Pygame开发的完整俄罗斯方块游戏。

## 功能特点

- ✅ 完整的俄罗斯方块游戏逻辑
- ✅ 7种经典方块形状（I, J, L, O, S, T, Z）
- ✅ 方块旋转和移动
- ✅ 行消除和得分系统
- ✅ 等级系统（随着等级提高，方块下落速度加快）
- ✅ 下一个方块预览
- ✅ 游戏暂停/继续功能
- ✅ 游戏结束检测
- ✅ 美观的游戏界面
- ✅ 完整的控制说明

## 游戏控制

- **← →** : 左右移动方块
- **↑** : 旋转方块
- **↓** : 加速下落
- **空格** : 硬降落（直接落到底部）
- **P** : 暂停/继续游戏
- **R** : 重新开始游戏
- **ESC** : 退出游戏

## 安装和运行

### 前提条件
- Python 3.7 或更高版本
- pip（Python包管理器）

### 安装步骤

1. 克隆或下载项目到本地
2. 进入项目目录：
   ```bash
   cd tetris_game
   ```
3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
   或者直接安装pygame：
   ```bash
   pip install pygame
   ```

### 运行游戏
```bash
python main.py
```

## 游戏规则

1. 方块从顶部下落，玩家可以控制方块的移动和旋转
2. 当一行被完全填满时，该行会被消除，玩家获得分数
3. 一次消除多行会获得额外分数
4. 一次消除4行（Tetris）会获得最高分数
5. 随着消除行数的增加，等级会提高，方块下落速度会加快
6. 当新方块无法放置时，游戏结束

## 分数系统

- 消除1行：100分 × 当前等级
- 消除2行：200分 × 当前等级
- 消除3行：300分 × 当前等级
- 消除4行（Tetris）：800分 × 当前等级
